## Learning Path

1. **basics/** - Start here to understand Java syntax
2. **controlflow/** - Learn decision making
3. **loops/** - Master iteration
4. **arrays/** - Work with collections of data
5. **methods/** - Organize code into reusable blocks
6. **oop/** - Understand classes, objects, and OOP principles
7. **strings/** - Master text manipulation
8. **exceptions/** - Handle errors gracefully
